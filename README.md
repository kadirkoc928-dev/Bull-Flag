# 🐂 Bull Flag Scanner (Streamlit + Python)

Scans US & EU Stocks for Bull Flag Breakouts using:

- yFinance
- Technical Indicators (RSI, ADX, MACD)
- Volume Confirmation
- ATR Risk Model

## Features

- Bull Flag Detection (Daily)
- Breakout Entry Calculation
- Stop Loss (Structure + ATR)
- Take Profit (2R / 3R / Projection)
- Streamlit Dashboard

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
